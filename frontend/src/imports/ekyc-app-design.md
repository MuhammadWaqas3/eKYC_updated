Design a premium, ultra-clean, Apple-inspired eKYC web application UI with a minimal, elegant and high-end aesthetic. The interface must feel calm, spacious, and refined. No heavy gradients, no excessive icons, no clutter. Focus on typography, whitespace, and subtle hierarchy.

Color Palette (Strictly Use):

Primary Background: #e7e4e4 (soft warm gray)

Primary Accent: #aa2771 (rich magenta/wine)

Secondary Text: #626262 (medium gray)

Card Background: #FFFFFF (pure white)

Text Primary: #1A1A1A (deep black)

Design Philosophy:

Generous white space

Large typography

Subtle dividers instead of borders

Very soft shadows (barely visible)

Smooth fade transitions

No unnecessary icons

Premium restraint

–––––––––––––––––––––––
TYPOGRAPHY SYSTEM

Primary Font:
Use SF Pro Display (if available) or Inter as fallback.

Hierarchy:

Page Heading: 32–36px, medium weight

Section Heading: 20–24px, medium weight

Body Text: 16px regular

Secondary Text: 14px, #626262

Buttons: 15–16px, medium weight

Letter spacing slightly tightened for headings.
Line height airy and comfortable.

–––––––––––––––––––––––
GLOBAL LAYOUT STRUCTURE

Minimal Header:

Top-left: Avanza logo (small, subtle).

Top-right: “Privacy” text link.

No heavy navigation bar.

Thin divider line (1px light gray).

Progress Indicator (Apple-style):

Thin horizontal progress bar.

Active portion in #aa2771.

Steps not displayed as icons.

Instead, show small text above bar:
“Step 2 of 5 – Identity Verification”

Percentage indicator:
Minimal text below heading:
“Verification 40% Complete”

No circular flashy graphics.

–––––––––––––––––––––––
STEP 1 – PERSONAL DETAILS

Centered white card (max width 520–600px).
Large heading:
“Let’s verify your identity.”

Subtle description text in #626262.

Instead of chatbot bubbles, use conversational form layout:

Floating label inputs

Thin underline style inputs (Apple style)

Focus state underline becomes #aa2771

Lock icon extremely subtle inside input field

Primary button:
Solid #aa2771 background
White text
Rounded corners (12–16px)
Soft hover effect (slight darken)

Tooltip:
“Why we need this?” as minimal text link.
On click → smooth expandable paragraph below.

–––––––––––––––––––––––
STEP 2 – CNIC UPLOAD

Large centered card.

Upload area:
Minimal bordered dropzone.
Light dashed border.
When active → border turns #aa2771.

After upload:
Replace dropzone with clean image preview.
Below it:

Heading:
“Review extracted details”

Fields displayed in elegant two-column grid.

Low-confidence fields:
Soft underline in #aa2771 (not red).
Small subtle helper text.

OCR Confidence:
Text only.
“Document confidence: 87%”

Retry:
Small text button:
“Retake Photo”

No harsh error states.

–––––––––––––––––––––––
STEP 3 – FACE VERIFICATION

Large minimal camera frame.
Soft rounded rectangle.

Overlay:
Thin oval face guide in subtle gray.

Side comparison:
Small CNIC image on side.
Live camera larger.

Matching result:
Large centered text:
“Face Verified”
or
“Verification In Progress”

Matching percentage:
Small secondary text below.

If issue:
Minimal text explanation:
“Lighting needs improvement.”
No red warning boxes.

–––––––––––––––––––––––
STEP 4 – FINGERPRINT

Minimal instruction text.
Large centered camera preview.
Thin outline hand guide.
No excessive UI.

–––––––––––––––––––––––
STEP 5 – CONFIRMATION

Full-width centered card.
Large heading:
“Confirm your information.”

Clean summary grid.
Editable fields subtle.
Primary button:
“Confirm & Continue”

After completion:
Centered success screen.
Large minimal checkmark.
Text:
“Verification Complete.”

Small secondary line:
“You’re all set.”

INTERACTION STYLE

300ms smooth fade transitions

Soft easing

No bounce animations

Micro scale on button hover (1.02)

Subtle shadow on card hover

–––––––––––––––––––––––
VISUAL RULES

No heavy gradients

No glossy effects

No neon glow

No loud success greens

Magenta only for focus + primary action

Everything else neutral

Overall Feeling:

Quiet luxury.
Confidence without shouting.
Technology that feels invisible.
